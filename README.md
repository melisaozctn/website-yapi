# Merkad Yapı Website

This project is a modern and responsive website developed for Merkad Yapı. It presents company information, services, images, and videos in a clean and user-friendly way.

## Features

Responsive design (mobile-friendly)
Interactive navigation menu
Image gallery with modal view
Video integration
Simple and fast structure

## Technologies Used

HTML5
CSS3
JavaScript

## Live Website

https://merkadyapi.com

## Project Structure

index.html
style.css
script.js

## Setup

Download or clone the project files and open the index.html file in your browser.
